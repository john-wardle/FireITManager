# FireIT Manager

Wildland Fire IT Management System for Camp Tracking

## Overview

FireIT Manager is a desktop application for modeling incident camp IT infrastructure,
inventory, personnel, and network topology during wildland fire operations.

## Getting Started

Install the project in editable mode and run the application from the repository root:

```powershell
python -m pip install -e .
python -m fireitmanager.app
```

Run the test suite with:

```powershell
python -m pytest
```

See [docs/README.md](docs/README.md) for the full documentation index.
